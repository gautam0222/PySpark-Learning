# 🚀 PySpark Learning on Databricks
commit

A comprehensive collection of PySpark concepts, transformations, actions, SQL operations, and Data Engineering fundamentals practiced using Databricks.

## 📖 Overview

This repository documents my hands-on journey learning PySpark and Databricks. It contains practical examples covering DataFrame operations, Spark SQL, schema management, transformations, aggregations, joins, window functions, UDFs, and data writing techniques commonly used in Data Engineering projects.

The exercises are performed using sample datasets such as Big Mart Sales and Drivers data within Databricks.
